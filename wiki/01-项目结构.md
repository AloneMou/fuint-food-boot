# 项目结构

## Maven 多模块

根 POM：[pom.xml](file:///d:/Project/Aite/Foot-Fuint-Backend-master/pom.xml)

- fuint-application：业务应用（Spring Boot 启动与各类 API）
- fuint-framework：基础框架与通用能力（web/redis/protection/common 等 starter）
- fuint-utils：通用工具类（字符串/属性读取等）

## 代码分层与职责（结合现有项目）

### API 层（Controller）

主要入口在：

- 管理端：`com.fuint.module.backendApi.controller.*`
- 会员端：`com.fuint.module.clientApi.controller.*`
- 商户端：`com.fuint.module.merchantApi.controller.*`
- OpenAPI v1：`com.fuint.openapi.v1.*`

返回模型存在两套风格：

- 管理端/会员端/商户端：继承 [BaseController](file:///d:/Project/Aite/Foot-Fuint-Backend-master/fuint-framework/fuint-common/src/main/java/com/fuint/framework/web/BaseController.java)，返回 [ResponseObject](file:///d:/Project/Aite/Foot-Fuint-Backend-master/fuint-framework/fuint-common/src/main/java/com/fuint/framework/web/ResponseObject.java)
- OpenAPI v1：返回 [CommonResult](file:///d:/Project/Aite/Foot-Fuint-Backend-master/fuint-framework/fuint-common/src/main/java/com/fuint/framework/pojo/CommonResult.java)，并配合签名/限流等注解（如 `@ApiSignature`、`@RateLimiter`）

### Service 层（业务逻辑）

典型接口与实现：

- 接口：`com.fuint.common.service.*`
- 实现：`com.fuint.common.service.impl.*`（常见 `*ServiceImpl extends ServiceImpl<Mapper, Model>`）

常见能力：

- MyBatis-Plus：`ServiceImpl`、`BaseMapper`、`LambdaQueryWrapper` 等
- 分页：`PaginationRequest` / `PaginationResponse`

### Repository 层（数据访问）

主要位置：

- Model（实体）：`com.fuint.repository.model.*`（常见 `Mt*` 命名，并带 `@TableName`、Lombok `@Getter/@Setter`）
- Mapper 接口：`com.fuint.repository.mapper.*`（`extends BaseMapper<T>`，可提供自定义方法与 default 方法）
- Mapper XML：`fuint-application/src/main/resources/mapper/*.xml`

### 公共模块（common）

`com.fuint.common.*` 包含：

- config：Spring 配置（Redis/MybatisPlus/Security/Swagger 等）
- enums：业务枚举（StatusEnum、PayTypeEnum 等）
- dto/param：API 交互 DTO 与入参对象
- util：项目业务工具（TokenUtil、DateUtil 等）
