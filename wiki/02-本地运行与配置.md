# 本地运行与配置

## 环境要求

- JDK 8（项目 `java.version=1.8`）
- Maven（根 POM 为多模块）
- MySQL（项目包含初始化 SQL）
- Redis（部分功能/会话依赖）

## 数据库初始化

- SQL 文件：[db/fuint-food.sql](file:///d:/Project/Aite/Foot-Fuint-Backend-master/db/fuint-food.sql)

建议新建一个本地数据库（例如 `fuint-food`），执行上述 SQL 进行初始化。

## 启动应用

启动入口：

- [fuintApplication.java](file:///d:/Project/Aite/Foot-Fuint-Backend-master/fuint-application/src/main/java/com/fuint/fuintApplication.java)

本地启动后，启动类会输出 Swagger 地址提示（默认示例：`http://localhost:8080/swagger-ui.html`）。

## 常用 Maven 命令

在仓库根目录执行：

```bash
# 编译（不跑测试）
mvn -f fuint-application/pom.xml -DskipTests package

# 运行测试
mvn -f fuint-application/pom.xml test
```
